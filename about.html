About the Project


Thank you to my advisor Professor Szymon Rusinkiewicz for supporting me in this project. And thank you to Professor Frederick Wherry for the conversation which inspired this entire concept.

The goal of my independent work project is to construct new methods of data visualization which I will use to investigate the nuance and inequality surrounding racial representation and experience on social media. I wanted to produce spiraling animations to map differences in social media representation by racial group. Through this project I would developed my own programs for combining different forms of data into a visual presentation. Taking statistics about top TikTok users I created independent shapes for each racial category, displaying differences in the data through aspects such as edge shape, color, and of course movement. The program is able to be applied to data on diverse subjects outside of social media representation.
